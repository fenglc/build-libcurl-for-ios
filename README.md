# build-libcurl-for-ios

## Usage

```
 curl -O http://curl.haxx.se/download/curl-7.45.0.tar.gz
 tar xf curl-7.45.0.tar.gz
 cd curl-7.45.0
 curl https://raw.githubusercontent.com/fenglc/build-libcurl-for-ios/master/build.sh | bash
 ......
```
